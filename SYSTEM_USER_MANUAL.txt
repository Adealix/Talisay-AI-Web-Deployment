PAGE A: ABOUT TALISAY (Learn about the fruit)
─────────────────────────────────────────────────────────────────────────────
Access: Click "ABOUT TALISAY" from home page or navigation menu

This page contains:

SECTION 1: FRUIT MATURITY GUIDE
─────────────────────────────────────────────────────────────────────────────
Detailed information about three maturity stages:

1. GREEN (Immature)
   • Appearance: Bright green color
   • Oil Content: 45-49% (lowest)
   • Best Use: Wait for maturity
   • Characteristics: Firm texture, bitter taste, not ready for oil extraction
   • Timeline: First 4-6 weeks of fruit development
   • Photo Carousel: Shows actual green fruits from research data

2. YELLOW (Mature / OPTIMAL FOR HARVESTING)
   • Appearance: Golden yellow color with slight browning
   • Oil Content: 57-60% (HIGHEST - BEST FOR OIL)
   • Best Use: HARVEST NOW for maximum oil yield
   • Characteristics: Soft texture, sweet aroma, peak oil content
   • Timeline: Weeks 8-10 of fruit development
   • Photo Carousel: Shows actual yellow fruits ready for harvest
   • Recommendation: This is the perfect harvest window

3. BROWN (Overripe)
   • Appearance: Dark brown color
   • Oil Content: 54-57% (still good, but lower than yellow)
   • Best Use: Can still extract oil, but lost some quality
   • Characteristics: Very soft, seeds are fully mature
   • Timeline: Weeks 11+ of fruit development
   • Photo Carousel: Shows actual brown fruits
   • Note: Harvest should have happened in yellow stage

SECTION 2: FRUIT CHARACTERISTICS
─────────────────────────────────────────────────────────────────────────────
• Scientific Name: Terminalia catappa
• Common Names: Talisay, Almond Tree, Tropical Almond
• Origin: Native to tropical regions

PART 9: TIPS & BEST PRACTICES FOR USER
===============================================================================

FOR BEST SCANNING RESULTS:
──────────────────────────────────────────────────────────────────────────────
1. Use good lighting - natural daylight is best
2. Include a ₱5 coin reference in the photo (for size calibration)
3. Make sure the fruit fills most of the frame
4. Avoid shadows on the fruit surface
5. Take multiple angles if possible

FOR ACCURATE ANALYSIS:
──────────────────────────────────────────────────────────────────────────────
1. Use clear, high-resolution images
2. Capture the whole fruit with all colors visible
3. Don't include other fruits if using single-fruit mode
4. Clean the phone camera lens before taking photos
5. Avoid blurry or out-of-focus images

OPTIMIZE YOUR EXPERIENCE:
──────────────────────────────────────────────────────────────────────────────
1. Save your analysis results for future comparison
2. Add notes to each scan for personal reference
3. Use the history filters to find specific analyses
4. Check "About Talisay" page to understand fruit stages better
5. Ask the chatbot if you have questions during scanning
6. Participate in forum to learn from other users

PART 10: TROUBLESHOOTING
===============================================================================

PROBLEM: Cannot log in with correct password
SOLUTION:
1. Check if caps lock is on (passwords are case-sensitive)
2. Try the "Forgot Password" option to reset
3. Ensure you're using the correct email address
4. Clear browser cache and try again

PROBLEM: Image upload fails or takes too long
SOLUTION:
1. Check your internet connection
2. Try a different image file
3. Compress the image size if it's very large (over 5MB)
4. Try uploading from camera instead of photo library
5. Check device storage - ensure enough space available

PROBLEM: Analysis results seem inaccurate
SOLUTION:
1. Try with a clearer image
2. Include a ₱5 coin reference in the photo
3. Ensure fruit is properly lit without shadows
4. Make sure it's actually a Talisay fruit
5. Try analyzing from different angle

PROBLEM: Cannot access certain features
SOLUTION:
1. Make sure you're logged in with valid account
2. Check if your account has admin rights (for admin features)
3. Refresh the page or restart the app
4. Try on a different browser or device
5. Contact system support if problem persists

PART 9: TROUBLESHOOTING FOR ADMINS
===============================================================================

PROBLEM: User cannot access their account
─────────────────────────────────────────────────────────────────────────────
STEPS:
1. Check user's account status (Active/Inactive/Suspended)
2. Verify email is verified
3. Check if password reset needed
4. Look at login error logs
5. Temporarily deactivate/reactivate account
6. If still failing, contact technical support with user ID

PROBLEM: Analysis results seem incorrect
─────────────────────────────────────────────────────────────────────────────
STEPS:
1. Check if correct ML model is deployed
2. Verify model accuracy metrics in Analytics
3. Check input image quality
4. Compare with research data baseline
5. May need to retrain model if systematic error

PROBLEM: High error rate in logs
─────────────────────────────────────────────────────────────────────────────
STEPS:
1. Check database connection status
2. Review API logs for patterns
3. Check server resources (CPU, memory)
4. Verify external API keys (Gemini, Cloudinary)
5. Restart affected service
6. If persists, contact infrastructure team

PROBLEM: Database is growing too large
─────────────────────────────────────────────────────────────────────────────
STEPS:
1. Run database optimization
2. Archive old analysis data (>1 year)
3. Delete old user sessions
4. Compress database
5. Check backup size - may need cleanup
6. Monitor growth rate

PROBLEM: User reports unable to upload images
─────────────────────────────────────────────────────────────────────────────
STEPS:
1. Check maximum upload size setting
2. Verify storage space available
3. Check user's upload limit (if enforced)
4. Review upload error logs
5. Test upload yourself to confirm
6. May be user's device/internet issue
